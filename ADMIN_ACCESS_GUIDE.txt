## Secure Admin Route
The admin panel is isolated for security and uses a completely separate layout.
**URL:** `/campusway-secure-admin`

## Default Admin Credentials
Default credentials can be rotated by setup scripts.
Use the latest values from `backend/INITIAL_ACCESS_INFO.txt`.

## Accessing the API
The admin API is secured behind a secret path.
**Secret Path:** `campusway-secure-admin`
**Full API Base:** `/api/campusway-secure-admin`

## Security Notes
1. **Change Password:** After logging in for the first time, change the password in the "Password" tab.
2. **Disable Default Account:** Create your own admin account and disable the default one for maximum security.
3. **Environment Variables:** In production, set a custom `ADMIN_SECRET_PATH` in your `.env` file.
